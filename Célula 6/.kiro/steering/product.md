# Product Overview

Sistema de procesamiento distribuido de trabajos (jobs) basado en AWS Lambda para procesar "toons" de forma asíncrona y escalable.

## Key Objectives

- Procesar lotes de toons de forma distribuida usando arquitectura serverless
- Mantener trazabilidad completa de jobs y resultados
- Garantizar idempotencia en el procesamiento
- Proveer APIs REST autenticadas para gestión de jobs y consulta de resultados
